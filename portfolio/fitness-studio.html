Projektöversikt

Project overview

Vi skapade en energifylld, mobil-först-upplevelse så att medlemmar hittar pass, tränare och bokar på några sekunder.

We built an energetic, mobile-first experience so members can find classes, trainers, and book in seconds.

  • Roll: UX, UI, bokningsflöde Role: UX, UI, booking flow
  • Tidsplan: 3 veckor Timeline: 3 weeks
  • Mål: Öka klassbokningar och lyfta tränarteamet Goal: Increase class bookings and highlight the trainer team

Viktiga funktioner

Key features

  • Veckoschema med filter för pass-typ och nivå Weekly schedule with filters for class type and level
  • Instruktörssidor med bio, certifikat och specialiteter Instructor pages with bios, certifications, and specialties
  • Bokningsformulär med önskad tid och betalningsplats Booking form with preferred time and payment placeholder
  • CTA-block för medlemskap, prova-på och PT CTA blocks for memberships, trials, and PT sessions
  • FAQ och platsinfo med kartplats för enkel navigation FAQ and location info with a map slot for easy navigation
